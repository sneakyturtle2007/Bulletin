API_Commands and their returns-
   login <username or email> <password> 
      - returns "User logged in" if login is successful, "Incorrect login" or "User not found" if login is unsuccessful
   
   creatuser <username> <email> <password> 
      - returns "User created" if user is created successfully, "User already exists" if user already exists

   deleteuser <username>
      - returns "User deleted" if user is deleted successfully. If error, then returns error message
   
   getuserinfo <username>
      - returns all information associated with the user

   createevent <username> <title> <date> <startTime> <endTime> <location> <publicityType> <invitees> <details>
      - returns "Event created" if event is created successfully, If error, then returns error message

   deleteevent <title>
      - returns "Event deleted" if event is deleted successfully. If error, then returns error message

   getevents <userid>
      - returns all events associated with the user
